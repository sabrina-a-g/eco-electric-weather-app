# Eco-Electric Weather App

A weather dashboard built with HTML, CSS, and Vanilla JavaScript.
Search any city to get live temperature, weather conditions, 
humidity, and wind speed. Toggle between °F and °C instantly.

## Features
- Live weather data from OpenWeatherMap API
- °F / °C temperature toggle
- Weather icons
- Error handling for invalid cities
- Responsive layout

## Built With
- HTML5
- CSS3 (Flexbox, CSS Variables)
- Vanilla JavaScript (fetch, async/await, DOM manipulation)

## What I Learned
- How to fetch data from a REST API
- Working with JSON responses
- Async/await and error handling in JavaScript
- CSS custom properties and responsive design

## Note
This project was built as a portfolio piece. The API key is 
visible in the source code — in a production app this would 
be handled server-side.
